────────────────────

REGISTRO (SignUp):
[✅] supabase.auth.signUp({email, password, options: {data: {...}}})
     → Crea usuario en Supabase
     → Envía email de confirmación
     → Guarda metadata (nombres, apellidos, CIP)

LOGIN (SignIn):
[✅] supabase.auth.signInWithPassword({email, password})
     → Verifica credenciales
     → Inicia sesión
     → Guarda token automáticamente

PASSWORD RECOVERY:
[✅] supabase.auth.resetPasswordForEmail(email)
     → Envía email de reset
     → Usuario hace click y resetea password

PROTECCIÓN DE RUTAS:
[✅] useAuth() en App.tsx
     → Verifica sesión al montar
     → Escucha cambios en tiempo real
     → Redirige a login si no hay usuario
     → Muestra dashboard si hay usuario

LOGOUT:
[✅] supabase.auth.signOut()
     → Cierra sesión
     → Limpia tokens
     → Redirige a login
