💡 CÓMO FUNCIONA AHORA

REGISTRO (Nuevo usuario)
─────────────────────────
1. Usuario completa formulario
2. Supabase crea la cuenta
3. Supabase envía email de confirmación
4. Usuario confirma haciendo click en el email
5. ✅ Cuenta lista para usar

LOGIN (Usuario existente)
──────────────────────────
1. Usuario ingresa email y contraseña
2. Supabase verifica
3. ✅ Sesión iniciada automáticamente
4. App redirige al dashboard

PROTECCIÓN DE RUTAS
────────────────────
- Si no hay sesión → Muestra login
- Si hay sesión → Muestra dashboard
- Todo automático con el hook useAuth()

📊 DATOS QUE GUARDA SUPABASE

Email + Password (encriptado)
Full Name (nombres)
Apellido Paterno
Apellido Materno
CIP
Email confirmado (true/false)
Fecha de creación
Fecha del último login

Acceso desde la app:
───────────────────
const { user } = useAuth()
user.email                          // Email del usuario
user.user_metadata.full_name        // Nombres
user.user_metadata.apellido_paterno // Apellido paterno
user.user_metadata.cip              // CIP
