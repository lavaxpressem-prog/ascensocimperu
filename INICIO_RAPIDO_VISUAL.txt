  STACK TECNOLÓGICO:
  ═════════════════
  • Frontend: React 18 + Vite + TanStack Router
  • Backend: Node.js + Express + Prisma
  • Database: SQLite
  • Auth: JWT + Argon2
  • Email: Nodemailer + Gmail


  PUERTOS:
  ═══════
  • Frontend: 5173
  • Backend:  3001
  • Cambiar en .env.local si es necesario


  CONFIGURACIÓN:
  ══════════════
  • Archivo: .env.local
  • Contiene: EMAIL_USER, EMAIL_PASS, JWT_SECRET, etc.
