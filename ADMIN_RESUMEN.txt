ESTADOS POSIBLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⏳ PENDING (Pendiente):
   • Usuario se registró
   • Espera aprobación del admin
   • ❌ NO PUEDE login
   • Admin puede: APROBAR o RECHAZAR

✅ APPROVED (Aprobado):
   • ✅ PUEDE hacer login
   • ✅ Acceso completo al sistema
   • Admin puede: SUSPENDER

🔒 SUSPENDED (Suspendido):
   • ❌ NO PUEDE hacer login
   • ❌ Acceso bloqueado
   • Admin puede: REACTIVAR
