PASO 1: TÚ CREAS CUENTA
  http://localhost:5173 → "Crear Cuenta"
  Completa datos → "Crear Cuenta"
  ✅ Eres ADMIN automáticamente

PASO 2: OTRO USUARIO CREA CUENTA
  http://localhost:5173 → "Crear Cuenta"
  Completa datos → "Crear Cuenta"
  ⏳ Status: PENDING (No puede login)

PASO 3: TÚ APRUEBAS AL USUARIO
  Login como admin → "Admin Panel"
  Ve usuarios pendientes → "APROBAR"
  ✅ Usuario ahora puede hacer login
